# docker-php7
Container with Ubuntu 16.04, PHP 7.1, MySQL & composer

## Install
First step, build the container « website »
```
./build.sh
```
And Run
```
./run.sh
```

## Commands
Start container « website »
```
docker start website
```
Stop container
```
docker stop website
```
Quit container
```
Ctrl+d
```
Delete container
```
sudo docker rm website
```
Delete image
```
docker rmi website
```
